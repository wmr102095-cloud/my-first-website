# Lecture1 프로젝트 설정

## 역할
- React + MUI 기반 웹 개발 학습 환경

## 문서 참조
@docs/design-system.md
@docs/code-convention.md
@docs/new_project.md

## 개발 환경
- Vite + React
- Material-UI (MUI) 완전 생태계
- React Router DOM
- Roboto 폰트

## 주의사항
- 새 프로젝트 시작 시 `_template_settings`를 복사하여 사용
- MUI 테마는 `src/theme.js`에서 관리
- ThemeProvider와 CssBaseline은 `src/main.jsx`에서 전역 적용
